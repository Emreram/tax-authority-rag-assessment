================================================================================
  NOTES
================================================================================

1. All ML models (embeddings, reranker, LLM) are self-hosted to ensure
   government data never leaves the national network boundary.

2. OpenSearch is the strategic choice because it unifies vector search,
   full-text search, and Document-Level Security in a single system.

3. Redis Stack is used exclusively for semantic caching, not as a primary
   data store. Cache is role-partitioned and has a conservative 0.97
   cosine similarity threshold.

4. If self-hosted LLM is not feasible, Azure OpenAI on Government Cloud
   (EU/NL region) is the approved cloud alternative with data residency
   guarantees.

5. Version numbers are minimums. Pin exact versions in production
   requirements.txt / pyproject.toml.
